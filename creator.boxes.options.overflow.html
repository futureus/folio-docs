Overflow Contents

By default, when the content is larger than the box layer, the content will be cut-off at the box edges. You can change this with the Allow contents to overflow control.

When this is set to "ON", the contents will display past the edges of the box so that you can see the full contents (or as much as can be displayed on the page). Note, however, that any interactivity (scrolling etc) will only work within the bounds of the box.
