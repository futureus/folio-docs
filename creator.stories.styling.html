Styling

Apply external style sheet

Text inside story layers can be styled in two different ways. Firstly, the basic mode of the story editor provides a WYSIWYG interface for the most common styling operations: bold, italic, underscore, strike-through, superscript, subscript, convert to upper case, capitalise, right align, left align, centre, ordered and unordered lists, font selection and font size and background, foreground and highlight colours, illustrated in "The basic mode story editor". For styling needs beyond those, it is necessary to go into advanced mode, and reference a CSS style sheet directly. Although possible to enter inline CSS, this is generally not the recommended approach, as such styles cannot easily be reused elsewhere. Instead, gather all your CSS styles in a separate file that you drop into your issue directory. Clicking the Style allows you to select the relevant style sheet file from your issue directory, as illustrated in "Apply external style sheet"
