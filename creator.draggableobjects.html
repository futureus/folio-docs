Draggable objects

Draggable objects in FutureFolio allow your user to interact with the page by dragging objects around the page. You can control which objects are draggable and when, you can also apply conditions to draggable objects to execute effects.

Be careful with animation scripts

You can run an animation script on a draggable object whilst a user is dragging it, however if you translate that object in a different direction to which they move it you may cause some undesired effects.

Respect the layer hierarchy

Draggle objects respect the layer hierarchy of the page, this means the user can drag an object behind another layer and release it. If they do this and you have not enabled pass-through-taps on that layer then they will not be able to interact with the draggable object again until the page reloads.
