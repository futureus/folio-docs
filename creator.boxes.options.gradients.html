Gradient Values

By default, no gradients are added to boxes and these fields can be left empty.

If you wish to add a gradient layer to a box, you can specify the start and end positions of the gradient and the direction. The values are entered as percentages relative to the size of the box. For a vertical gradient the scale goes form 0% (top) to 100% (middle) and back to 0% (bottom), hence a start value of 20% and end value of 50% would place a gradient over the top 1/5 of the box and the bottom 1/2. When working with horizontal gradients the values work the same but from left to right.
