Print A File

This behaviour lets you attach a PDF or high-resolution JPEG/PNG to your issue that can be printed out when the user activates the behaviour. For example, your page could discuss some guitar tablature or stitching patterns, then have the actual data printed when the user activates this behaviour.

Parameter: the filename to print. Please ensure your PDF has no encryption attached.

AirPrint is required

This feature requires that the user have an iOS-compatible printer.
