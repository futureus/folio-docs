Long Pages

Sometimes a regular length page isn't going to cut it, you might have a long flowing article where you want multiple images and elements intersecting the copy. Normally something like this would have to be done in HTML and CSS via a Story Layer, however if you want to add true interactive elements to a page set out like that, it just isn't possible. The only way to achieve these features is via a custom sized page.

A custom (or long) page extends the canvas size of your page, this enables you to use the FutureFolio creator to layout the entire page from top to bottom. Users will be able to scroll all around this large canvas and see it just as you have laid it out.

Backgrounds images don't scroll

With long pages the background will not scroll with the page. It remains static in it's original position.
