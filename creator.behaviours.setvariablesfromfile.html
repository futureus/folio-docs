Set Variables From File

This lets you set multiple variables at once, and is usually to create lists easily. To use this behaviour, start by creating a file in your editing issue directory that contains the variables and values you want to set. For example, if you wanted to add the value "Ground beef" to the variable "$Meat", you would do this:

$Meat
Ground beef

If you want to set multiple values at once, just list them one per line, like this:

$Meat
Ground beef
Chicken drumsticks

You can also set multiple variables at once by listing their names as needed, like this:

$Meat
Ground beef
Chicken

$Dairy
Cheese
Milk

That will set the $Meat variable to be a list containing "Ground beef" and "Chicken", and set the $Dairy variable to be a list containing "Cheese" and "Milk".

Once you have your variables file created in your editing issue directory, use its filename as the parameter to this behaviour. If you want to remove the variables rather than set them, prefix your filename with a minus symbol, eg -myvariables.var.

Parameter: the filename containing the variables to set.
