Page settings

With the pages tab selected, select a page and click on the triangle icon on the right of the page layer cell, select settings.

Within this settings panel you can set the following attributes for the page:

  • Name - the name of the page as it appears in your list of pages.
  • Title - this is the title for the page that a user will see when the page appears in search results, either inside or outside or the issue.
  • Description - a brief description for the page. This description is searchable and will show in search results.
  • Share URL - a URL that can be shared by Facebook or Twitter. This is unique to the page and not the issue as a whole.
  • Page type - by default pages are set as editorial. The default configuration for an editorial page is to show the chrome and to appear in the table of contents (TOC). Marking a page as an advert page changes the default settings. An advert page is configured to hide the chrome and not appear in the TOC. You can overwrite either of these default settings to achieve other configurations.
  • Force chrome - setting this to show or hide will set the chrome for that page respectively, irrespective of the page type.
  • Force TOC - setting this to show or hide will set wether that page appears in the TOC, irrespective of the page type.
