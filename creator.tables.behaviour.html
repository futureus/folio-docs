Row Behaviours

Tables can perform an action in response to a user tapping on a row.

Each row contains a behaviour button which you can use to assign a behaviour to that row. The default behaviour is "Do nothing", so the button will initially display this.

To set a behaviour to the row, tap on the behaviour button and select a behaviour from the menu (see "Behaviours" for information on each behaviour type).

Some behaviours require a data parameter to be specified - e.g. Jump to page will require the name of the page to jump to. Specify this data in the text field to the right of the behaviour button.
