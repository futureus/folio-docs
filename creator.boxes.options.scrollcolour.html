Scrollbar Colours

When the contents of a box extend past the edges of the box itself, you can set the box up so that the user can scroll to see the overflow contents. (See "Scrolling Box Contents")

As the user scrolls, scrollbars appear on the edges of the box. By default, these scrollbars are black, but you can change them to be white or turn them off entirely.

To change the scrollbar colours, select the box layer, click on the Layer settings button and choose the required option on the Scrollbars control.

Scrollbar colours - black, white, none
