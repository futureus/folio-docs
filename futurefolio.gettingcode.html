Getting a code

FutureFolio Viewer enables you to download issues using a unique code. To get a code you need to contact Digital Editions digital.editions@futurenet.com with your publication name, it's issue number and issue month. Please note that Digital Editions can only provide codes for completed and generated issues. Please check with the Pre Press department if you are unsure if your issue has been generated.

With your code in hand, tap the Enter URL button and enter the code. Tap Go and the issue will start to download. Once complete the issue will appear in the library area.
