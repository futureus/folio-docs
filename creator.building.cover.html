Cover page

The build process creates an image file to use as your issue's cover page when it is viewed within the user's FutureFolio library of issues. By default, this is created from the first page in the issue.

You may wish to use a different page - for example, if you have a cover-wrap advert as the first page in your issue.

To specify a different cover page for your issue, enter the page's name in the Cover page field. If you have entered an invalid name, a warning will be displayed when you start the build, or dismiss the build window. The first page will be used as the default in this situation.
