Boxes

This section covers the "box" layer type and how to use it in your issues.

Box layers are used to group layers which contain content together and control them as a set. They are also used to provide extra behavioural and display features to the content layers.

The areas covered are:

  1. Grouping Elements - using boxes to organise your page.
  2. Scrolling Box Contents - defining how the user navigates content in a box.
  3. Box Options - customising your box.
  4. Passing-through Box Taps - allowing your user to tap an object under your box.
