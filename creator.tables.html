Tables

Table layers are used to display data and / or images in a user-sortable table.

This section covers creating a basic table layer and the various customisations you can then apply to it:

  1. Table Creation - how to create, duplicate and delete a table.
  2. Sections, Columns and Rows - laying out the skeleton table.
  3. Data Entry - entering cell data and the special formats supported.
  4. Row Behaviours - adding behaviours for user row taps.
  5. Customising Your Table - adding colour and background images to your table.
