Hide Parent Layer

When a user taps an image that has this behaviour attached, the images parent layer will be hidden. This is always used in conjunction with boxes, and allows you to show a box and hide it with very simple interactivity.

A common use for this behaviour is to have a box with multiple layers inside, such as a slideshow, captions for the pictures, and a close button. Rather than have the close button hide just itself (which is what Hide This Layer would do), using Hide Parent Layer would cause the whole box to be hidden, hiding the slideshow, captions and close button all at once.

Parameter: Not used.
