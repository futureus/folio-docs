Editing in FutureFolio Editor

FutureFolio editions have been designed to allow editing from multiple sources, of which one is the FutureFolio Creator tool that you should by now be familiar with. We have a second tool, called FutureFolio Editor, that allows you to edit a small subset of your page's features using a simplified user interace. While Creator is the preferred tool for all layout and previewing work, Editor is the preferred tool for making copy and image changes. In fact, many magazines have been designed in such a way that production staff only need to use FutureFolio Editor to modify templates.
