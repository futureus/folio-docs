Grouping Elements

If you have several layers which logically belong together, you may wish to combine them as a group, using a box layer. This then enables you to manipulate them as a whole, saving you from performing many repeated actions.

For example, you may have several images and a story which you want to be initially hidden, showing as a result of the user tapping a button. To achieve this without unnecessary repetition, you would place the image and story layers inside a box layer, hide the box layer, then add a tap effect to the relevant button to show the box layer.

To add a box, click on the Add box button

Adding a box

Once you have a box layer, you can place a layer inside it by selecting that layer, clicking the Place Inside button (which will turn blue), then selecting the box layer in the preview window. Note that you must first select the layer to be grouped, then the box layer you want it to be placed into. Should you click on an item that is not a box layer, it will automatically place it inside that layer's encapsulating box.

The Place Inside button

If the action was successful, the layer will now be listed under the box layer and slightly indented.

Layer successfully placed inside box

Once your layers are grouped within a box, you can then use the box to manipulate the set of layers, both in the creator and through effects. For example, moving, animating, hiding, performing tap effects on the box layer etc, will also affect its grouped layers.

Layer positions

Note that once a layer is nested inside a box, its position becomes relative to the box rather than the canvas.
