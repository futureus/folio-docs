Fit vs Stretch

The Size button lets you choose the fit-mode of your image in the image layer.

Fit mode honours the aspect ratio of the source image(s). For each image, if the image layer is a different aspect ratio, the image will be displayed as large as possible while still maintaining the aspect ratio via letterboxing or pillarboxing.

Stretch mode forces the image to fill the image layer, disregarding the aspect ratio.

Size mode combined with rotation and scale

To effectively combine size mode changes with non-default rotation and scale settings, it is important to understand how they work together.

The first setting which is applied is the size mode - fit or stretch. So your image is sized within the image layer according to your fit / stretch setting before any rotation or scale is applied.

Lastly, the rotation and scale is applied in one transformation. This transformation is applied to the result of placing the image within the layer using the given size mode.
