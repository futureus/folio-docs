Interactive editions

If you're looking to create a magazine specifically for tablets and smartphones, you'll need to use FutureFolio's Creator tool. This is an in-house software package that lets you layout pages with interactivity, test what you've made, and package up the results ready for distribution.

FutureFolio Creator is a large and powerful tool, with many options that let you get your pages working exactly as you want. If this is your first time using the software, please start at "Quick start".
