Reading the tilt sensor

You can create some interesting page effects, not least faux depth, by enabling the device's accelerometer. This will read the user's movement of the device as they tilt it left, right, forwards and backwards, and send that movement data to your page so that layers are shifted to give the user a different perspective on the page.

To enable the accelerometer for a page, use the behaviour "Enable accelerometer" either as a rendition behaviour or an image behaviour and give it a number for its parameter. This number affects the speed that should be shifted as the user moves their device, and it defaults to 1. You can set large values (eg 10) to allow the user to reveal things by moving their device.

When the "Enable accelerometer" behaviour has been triggered, all layers on the current page that have Dynamic Movement enabled will respond to the user's tilting actions.

FutureFolio will move each object at varying speeds depending on their position on your page: objects at the back of the page will move slowest and objects at the front will move fastest.

Objects inside boxes will move inside the box, even if the box itself doesn't have dynamic movement enabled. The speed of each object is determined by the position of its parent box, and its position inside its parent box.

Be careful with the accelerometer

Please read the following provisos with tilt behaviours:

  • Once enabled, the accelerometer can't be disabled for that page unless the user scrolls a couple of pages away and returns.
  • The accelerometer will move any dynamic movement layers; if you try to animate or use physics on these layers while they are moving, you will not get the effect you want. Do not try to animate or use physics with tilt-enabled objects once the accelerometer has been enabled for the page.
  • Only one page can have control over the accelerometer at a time. If you intend to have two adjacent pages that use the accelerometer, make sure you place an image on the page that allows the user to re-activate the accelerometer for whichever page they choose.
