Creating templates

Templates are the bread and butter of issue creation, they shave hours off creation time and can be a life saver should you need to create duplicate pages.

To create a template first you need to create a page in FutureFolio Creator. Place your elements and add constant graphics, like background, brand logos and repeated artwork. Once you have a functional page, return to the Issue Editor window and tap the right arrow next to this new page and tap "Make Template".

Your new template will now reside in the central template location. You can add a page of that style to any issue you are working on, templates are (as mentioned in the previous section) global.

Avoid images for global templates

If you plan on using your template in other publications, you may not have the correct image assets in that title, try and avoid including images in this case.

If you want to share this template with others, you will need to send them the template file. Template files are located in /Simulator/Settings/. Inside this folder you will find multiple .grid files.
