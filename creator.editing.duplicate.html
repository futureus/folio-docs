Duplicating layers

If you wish to produce a copy of a layer on the page, first select it then tap the Duplicate button. Note that duplicated layers are identical to their originals: text retains its styling, images retain their behaviours.

Duplicating tables

Tables have their own data source saved as an external file. When you duplicate a table, Creator automatically creates a duplicate of the data file as well, meaning that your tables will point at different data - if one changes the other will not update.

If you want both tables to point at the same data, you can modify one of the tables to point at the data of the other.
