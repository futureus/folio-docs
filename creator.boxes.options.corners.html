Rounded Corners

By default, layers have sharp corners. Using the Round corners control on a box layer, you can give it rounded corners. If the layer to be rounded is not a box layer, you can group it in a box layer (see "Grouping Elements") and apply the setting to that.

To round the corners, type in the required amount in the numerical entry field, e.g. 5, 20, 50 would give the results shown in the figure below.
