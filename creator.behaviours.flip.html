Transition: Flip Horizontal / Vertical

Flip animations allow you to hide one view while showing another, which gives the impression of a coin flip on the page. This is commonly used to show that two layers are attached somehow, such as a question and answer.

The basic way to use a flip is to specify the name of the layer to flip to as the behaviour's parameter. For example, if you wanted Image A to flip to Image B, you would attach the behaviour "Transition: Flip Horizontal" to Image A then give it the parameter Image B.

An alternative way to use transitions is to specify to layer names as the parameter, separated by commas. This will flip from the first layer to the second, without necessarily affecting the layer that has the behaviour attached.

Parameter: Either the name of the layer to flip to (in which case, it's assumed the current image is the one to flip from), or the names of the layer to flip from and flip to, separated by commas. If multiple layers have the same name, they will all be flipped.
