Images

Image layers are used to display images with various types of interactivity for the user. The layer type ranges from a simple standalone image to an interactive 360-degree swipeable view and many more besides.

To create a new image layer, click on the Add image button.

Adding an image

Included in this section is:

  1. Importing Filenames - specifying which image(s) to display.
  2. Slideshow - a series of images which the user can swipe through.
  3. Animation - a series of images which automatically play back in sequence with no user interaction.
  4. 360 - images of an object from all the way around which the user swipes through, giving the appearance of a scrollable 3D model.
  5. Scratch - a top layer image which the user can scratch away to reveal a bottom layer image.
  6. Zoom - a pinch to zoom image area.
  7. Wipe - a top layer image which the user can swipe to and fro over a bottom layer image.
  8. Rotation and scale - transformations which can be applied to image layers.
  9. Fit vs Stretch - image display mode within the layer.
  10. Flip mode - flipped image display.
