Jump To Slideshow Page

This behaviour lets you move an image slideshow to a specified page by specifying its layer name and page number separated by commas.

For example, to make an image slideshow called "MySlideshow" jump to image number 8, you would use the parameter "MySlideshow, 8".

Parameter: a comma-separated value containing the name of the sideshow to change and the page number to show.
