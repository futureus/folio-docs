Converting to templates

If you've found a problem with your layout and have multiple pages using it, you would normally have to manually change each page to update the layout. However you can instead change one page, create a template from that and use the "Convert to Template" button to auto apply layout changes to your page.

Example of a template conversion success

To utilise the "Convert to Template" function you will need a template with the same number of layers all matching in type and name, everything else can change.

If your templates do not match then the convert will fail and you will be presented with a list of differences and two conversion options:

  1. Ignore extra layers - This will leave all layers that do not match intact and not move them.
  2. Delete extra layers - This will remove the layers outlined leaving only the ones that match.

Conversion error, missing layers

"Convert to Template" can also be used to repurpose content from other FutureFolio publications with little effort.
