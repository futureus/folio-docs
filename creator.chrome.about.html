What is a chrome?

A chrome is an overlaid toolbar that enables the user to navigate your issue easily, access extra features and close that particular issue.

The key to a good chrome is to keep it clean and simple to use and be sure to implement both a navigation method and a Close Issue button, the rest is down to the publication.

Chromes are generated by creating and editing the chrome.xml file. Your chrome can load images from your issue directory and you can have multiple instances for different renditions and devices.
