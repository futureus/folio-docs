Copy a file to iTunes

This behaviour allows you to copy one or more file from your issue into the iTunes Documents directory for your app. Your readers can then connect their device to iTunes and drag the files from there onto their PC.

Only files that you specify will be copied. If the file exists already, it will be overwritten.

Parameter: the filename in your issue that you want to copy. If you want to copy multiple files at the same time, you can specify them as a comma-separated list. It is recommended that you take this approach rather than use multiple behaviours that copy single files, because this latter option will show multiple messages to the user.
