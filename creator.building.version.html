Issue version support

Each version of FutureFolio Creator usually has a host of new features not available in the previous version. Thus the FutureFolio apps available on the App Store are updated inline with these version increases.

To prevent users from encountering a feature that is not supported in their version of the app, the build process places a numeric internal "Folio version" code inside the main issue data that informs the app what feature support it needs. If the app cannot support the issue it's trying to read it will inform the user that they need to update it in order to view this issue.

What version is your FutureFolio app?

To find out the version support of your App Store app, please ask your FutureFolio contact. They will be able to inform you if your app is indeed up to date or if you require an App Store update submission.
