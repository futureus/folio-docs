Keyboard Shortcuts

The following keyboard shortcuts can be used inside the page editor window:

  • b: Send layer backwards.
  • f: Bring layer forwards.
  • g: Toggle Grid Mode
  • l or Spacebar: Toggle Locked Mode.
  • p: Toggle Place Inside.
  • r: Rename layer.
  • s: Show rendition settings.
  • z: Undo.
  • Shift+z: Redo.
  • + or =: Increase page zoom.
  • -: Decrease page zoom.
  • 0: Return to original page zoom.
  • /: Toggles between the original page zoom and your current page zoom.

As well as this, using Ctrl+Alt+Up/Down/Left/Right will move the selected layer by one pixel each time.
