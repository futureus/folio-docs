Animation Scripts

Animation scripts have a precise, XML-based file format. There is no tool for editing them at this time, so you'll need to edit the XML by hand in your preferred text editor. While this isn't much fun, the results that can be obtained make it worth while!

Before we start, there are some key terms you need to know:

  • Alpha - the opacity of a layere, where 1 is 100% opaque and 0 is 0% opaque. Can be partial values, such as 0.5
  • Angle - the rotation of a layer on the page, expressed in degrees, where 0 its its normal rotation and 90 is rotated 90 degrees clockwise.
  • Animation - one or more timed movements.
  • Curve - how movement accelerates and decelerates. Can be one of "start", "end", "both" and "linear".
  • Delay - the number of seconds an animation will pause before playing.
  • Duration - the number of seconds an animation takes to happen. Can be partial values.
  • Layer - a single movement inside an animation.
  • Scale - the size of a layer on the page, where 1 is 1x its normal size (100%) and 5 is 5x its normal size (500%). Can be partial values.
  • Sequence - chained animations that run in a particular order.
  • X/Y - the position of a layer on the page.

Each animation can contain multiple layer movements, all of which will execute with the same duration and delay.

Each layer movement can affect multiple layers on the page, either because you specified multiple layer names or because several layers have the same name.

All transformations are drawn on the device's graphics card, which is incredibly fast - this allows you to do some fairly advanced effects without worrying about performance.

All values specify where you want a layer to end up, rather than where it's coming from. So, if a layer has scale 2 and you set it to have scale 1, it will scale down; if it has scale 0 then it will scale up. Don't worry about where it is, just specify where you want it to be.

The secret to creating animations is that you only need specify the values you want to change. For example, if you want to make a layer fade out, set its alpha to be 0 - you don't need to set its X, Y, angle or scale values, because they aren't changing.
