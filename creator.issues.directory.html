Issue Directory

Each issue has its own unique directory which can be found inside /Simulator/EditingIssues. All of the assets for that issue should be placed inside the issue's folder. FutureFolio Creator will make this directory for you when you add an issue.

To add an issue, click on Add New Issue -> Create blank issue: "Adding an issue"

If you have created any issue templates, they will be listed in the menu and you can create the new issue from these by selecting the required one.

Adding an issue

To delete an issue, click on Edit then click on the cross for the relevant issue: "Deleting an issue"

To return to the normal issue view once you have finished deleting, click on Done
