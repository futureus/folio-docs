Launching a HD page

If a fully interactive issue is just unsustainable, but you would like to add the odd fully interactive page here and there you can do this by creating a single page issue (see "Interactive editions") and including it as a folder in your issue archive.

To link this fully interactive page up, create a hyperlink and point it at your folder using the following style: media://myFolderName.

Leave the folder structure intact

When embedding a HD page, you need to link to the folder that contains that page's assets, inside that folder should reside a meta folder. Linking to the incorrect location or playing with the folder's internal layout may result in a dead link.
