Hosting an issue

This should only be used internally

Hosting issues uses bandwidth, this costs the company. Only use this feature for client and internal sharing of issues.

Using a web server or online service you can host your issue file and provide the file's URL to FutureFolio Viewer users, they can enter that URL in the Enter URL pop over and download that issue. This is great for sending WIP issues to clients for approval.
