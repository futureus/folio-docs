Routes

Add a route

The Creator map layer supports route overlays. A route a track imported from a GPS device, or software that supports the standard GPX format. Routes can be used to describe for example how to navigate from one position to another, or the description of a race or other event. GPX is a widely supported XML standard to describe coordinate data. Creator supports routes represented using the track segment of a GPX file only. This is the normal way that a gps device would record a track. However, there are other parts of the GPX format that Creator will silently ignore, notably way points. In other words, Creator will show the actual track, but not show any extras, such as points of special interest, that might be embedded in the GPX file.

To import a route from a GPX file, click on the Route text field to select a GPX file to import ( "Add a route"). Note that you may need to scroll and zoom the map to see the route, if the route's extent is far away from the current map centre.

GPX format compatibility

A GPX file can represent a route in several ways, using the , and the tags. FutureFolio only supports the tag, which is the most commonly used in GPS devices.
