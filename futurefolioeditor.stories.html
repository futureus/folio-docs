Editing Stories

FutureFolio Creator lets you modify the contents of story elements by typing into three fields: headline, strap and body. To edit a story layer in FutureFolio Editor, double-click on the layer name to bring up the editor window. You can type either using HTML (writing tags as you normally would) or just by using the keyboard shortcuts Cmd+B (toggle bold), Cmd+I (toggle italics) and Cmd+U (toggle underline).

If you add styles to your text using the keyboard shortcuts, you should press the "HTML" button before saving to have your styles converted to HTML. This should only be done once, otherwise Editor will try to HTML encode something that has already been HTML encoded.
