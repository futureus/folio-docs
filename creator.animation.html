Animation

Animation scripts in FutureFolio allow you to make layers move around the page, zoom in or out, fade in or out, or spin – or all those things at once. You can set a delay and duration for each animation, animate multiple layers simultaneously, and even create animation chains that allow animations to be executed one after another.

Animation scripts must be created by hand in a text editor, then linked into behaviours using their filename.
