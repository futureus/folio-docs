Show Single Box Layer

Show Single Box Layer works in the same way as Show Single Layer, but it restricts its effects to layers contained in the same box as the owner of this behaviour.

For example, if you create ImageA and give it the behaviour Show Single Layer with the parameter "ImageB", FutureFolio will find the layer ImageB wherever it exists on the page and show it, hiding any other layers on the page that were created invisible. If you use Show Single Box Layer instead, the only layers that will be shown or hidden are the ones inside the same box as ImageA. If ImageA is not inside a box, then it will only affect layers that are also not inside boxes.

Show Single Box Layer is used only when you have multiple Show Single Layer effects on a page and you want to avoid them clashing.

Parameter: the name of the layer to show. If you wish to show more than one simultaneously, separate their name with commas.
