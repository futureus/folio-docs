Relative values

Animations values can be manipulated relatively, the following attributes can be altered based on their current value on screen:

  • Alpha
  • Angle
  • Scale
  • X/Y

To use a relative value you need to reference the existing value, this is a matter of entering the name of the attribute inside the new value and one of the following operand:

  • + - This adds to the value before it. so x+10 would place the layer at it's current x position only 10 pixels to the right.
  • - - The opposite of +, so x-10 would move the layer left 10 pixels, whileangle-10 would rotate the layer 10 degrees counter-clockwise.
  • * - Believe it or not, this is the times symbol, so x*1.5 would increase the x value of the layer by 50% of it's x value.
  • / - Finally you can divide the value, so y/1.2 would reduce it's y position by about 83%.
