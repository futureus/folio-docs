Auto Format

Apply Auto Format

When flowing in body copy from an external text file not already marked up as HTML, Creator allows a degree of conversion using its Auto Format options, as indicated by "Apply Auto Format". This will allow links to be properly respected, and also inserting HTML paragraph breaks where the copy contains empty lines. Note that there are limits as to how successful such a conversion will be. Some subsequent manual checking and editing may be required.
