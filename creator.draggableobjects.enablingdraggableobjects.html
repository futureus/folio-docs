Enabling draggable objects

Any object you wish to be draggable at run-time must have its "Dynamic movement" property set to ON.

To enable all draggable objects on a page add "Enable draggable objects" to the pages renditions setting panel and set the "Plays" parameter to "1".

To enable draggable objects as the action of a tap effect, add "Enable draggable objects" as a tap effect to a layer. The "Value" parameter specifies the layer names that you wish to become draggable (comma separated), and again the "Plays" parameter must be set to "1".

Draggable objects can not be disabled

Once enable draggable objects will remain draggable until the page is unloaded.
