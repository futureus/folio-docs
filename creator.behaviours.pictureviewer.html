Open Picture Viewer

New feature

This feature won't be available until the next release

This behaviour will show a full screen picture viewer which supports pinch-to-zoom and double-tap-zoom.

Parameter: the list of images to display, separated by a comma, e.g. image1.jpg, image2.jpg, image3.jpg

Note that the images specified to be displayed in the picture viewer will not be optimised down in size by the build process. This is to enable good resolution at the point when the user zooms the image. This means that you need to be careful not to include too many overly large images, so as not to overload the device.
