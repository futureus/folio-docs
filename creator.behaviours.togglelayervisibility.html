Toggle Layer Visibility

When activated, this will show or hide a named layer. If the layer is hidden, it will be shown; if it is showing, it will be hidden.

This behaviour allows you to show multiple layers at a time simply by toggling them on with multiple behaviours.

Parameter: the name of the layer to toggle. If multiple layers have the same name, they will all be toggled.
