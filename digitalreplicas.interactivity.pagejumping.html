Page jumping

Publishers can create an interactive reading experience by linking users to later pages. Using this you can make cover hits jump to the first page of that article or an index page filled with clickable links to start of each section.

To creating a page jump link, add a hyperlink to a region of your PDF and set the url to page://10 (replacing 10 with the page number you want the issue to jump to).

Page numbering and you

The desired page number needs to be actual number from the front cover onwards with the front cover being page one.

Page numbering mismatches

Don't use the page-0.pdf filename numbering for your jump to page hyperlink, the filenames start at 0 while your actual page numbers start at 1. So in the case of page-0.pdf, this is actually page://1
