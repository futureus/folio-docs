Advanced Pages

Sometimes you'll want to go beyond the normal issue setup; create long pages, stacked vertical browsing or multiple layouts for different devices. These can all be achieve within FutureFolio Creator.

Here's a breakdown and brief description of each of the advanced page features:

  1. Renditions - Renditions decide what automated layout and device your issue supports. You can create multiple renditions for each page.
  2. Long Pages - Change the default size of the page, users can then scroll around it.
  3. Stacks - Enable users to browse pages vertically as well as horizontally.
