Play a Movie

If you specify the name of a movie layer as this behaviour's parameter, that movie will be played wherever it appears on the page as if the reader had tapped its play button. If you specify a filename of a local movie or a URL to a streamed movie, the current page will be hidden and replaced with a full-screen movie view.

If you want to allow the user to rotate their device to watch the movie in any orientation they want, you should specify a filename as the parameter to this behaviour.

Parameter: either the layer name of the movie to play, or the URL to stream.

If you are using the behaviour "Download all streamed movies" in your issue, you may wish to supply a different URL to download the movie from. For example, you may wish to stream an SD version, but allow the user to download an HD version. To specify a separate download URL, enter it after your streamed URL in the parameter field, separated by a comma.

E.g. http://streammoviefrom.mp4, http://downloadmoviefrom.mp4
