Working with Angles

Any change of angle will be animated using the shortest possible change. For example, if you specify an angle of 270 and the layer currently has an angle of 0, it will rotate 90 degrees anti-clockwise rather than 270 degrees clockwise

By default, rotation of an object occurs around its centre point; see "Anchor Points" for how to modify the point of rotation.
