Options

The map layer screen has a number of options that you can use to change various aspects of the presentation of the map. The Scroll switch determines if the user can move the map around by dragging it. Similarly, the Zoom switch is used to allow or disallow the map to be zoomed in or out to give either a more detailed, or a more overview look. With both scroll and zoom switched off, the map itself will appear static to the user. Switching on Show user will make your map show the user's current position on the map using the standard iOS pulsating blue dot.

The map can be shown in three different forms, as set by the segmented control at the top of the options section: Standard, Satellite or Hybrid. These are your normal Google Maps map types. Standard is the default setting, being a normal, drawn map. The satellite option shows satellite imagery which can be zoomed in to a varying level depending on location. The hybrid type is a combination of the other two, where the satellite imagery has the drawn map overlaid.
