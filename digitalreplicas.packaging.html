Packaging your issue

All issues in a FutureFolio container app are downloaded as archived .zip files. For any issue you've created to be downloadable or viewable in a FutureFolio container app you will need to archive your issue files.

Simply place all your PDF, thumbnail, video, sound and image files in a single location along with any directories for fully interactive counterparts. Then select these files (not the folder) and archive them using: File > Compress X items.

This archive is your full issue, you can now test this with the FutureFolio Viewer (see "Viewing in FutureFolio Viewer"). Once you are happy, upload this archive to the filetransfer.future.net.uk server using the credentials supplied to you when your account was first set up.
