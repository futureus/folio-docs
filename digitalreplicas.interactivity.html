Adding interactivity

Issues created using the Digital replica technique can too contain interactive elements like the fully featured Interactive editions. These features are restricted to a set list of possible interactive elements (Page jumping, Movies, Sound, Images and HD pages) and are added to your issues through creating link boxes in your PDF files.

The interactive features covered in this section are as follows:

  • Page jumping - Create fully interactive contents pages that jump the user to the correct location.
  • Showing a slideshow - Display images and image slideshows without adding extra pages to your issue.
  • Playing video - Play videos from within the app.
  • Playing audio - You can play music and noises in your issue.
  • Launching a HD page - Embed an Interactive edition page within your Digital replica.

Depending on your desktop publishing tool of choice you will most likely be able to add custom hyperlink boxes to your documents. This is normally done by creating a custom sized box element and adding the link to the entire entity.

For example in Adobe InDesign you need to create an invisible box object and add the link to that via the Hyperlinks pane. To show the Hyperlinks pane click Window > Interactive > Hyperlinks, you can then click "New Hyperlink" to add the clickable link.

All URLs are case-sensitive

Please enter all URLs correctly, iPhone and iPad devices are case sensitive and will fail to load interactivity if any filenames are entered incorrectly.

No extra characters!

If your URL has a space on the end or a return character it will not work. FutureFolio simply reads the link you've provided it doesn't manipulate it. Be sure to check your issues using the FutureFolio Viewer if you are unsure.
