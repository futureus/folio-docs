Email a Variable

This behaviour lets your reader send the contents of one or more variable to someone by email. For example, if you had made a wish list of games they want to buy, setting the value of this behaviour to be $Wishlist would paste the contents of the wishlist into an email. You can specify multiple variable names, separated by commas, in which case FutureFolio will paste all of them into the email, separated by the titles of the variables in bold.

NB: The email window will only be shown if there is some content to send - if all the variables you specify are empty, nothing will happen.

Parameter: the name(s) of the variables to email.
