Download All Streamed Movies

This behaviour causes all streamed movies in the issue to be downloaded into the issue's folder on the device. When the user then plays back the movies, they play from the local file. This allows issue sizes to be small, while also allowing the user to watch the movies while offline.

Note that when specifying the URL to stream a given movie from, you can also supply a different URL to download it from when this behaviour is triggered - for example, so that you can stream an SD version and download an HD one. See the individual sections on playing movies for details on how to do this.

Parameter: not used.
