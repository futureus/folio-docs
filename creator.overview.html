Overview

This section will walk you through the basics of FutureFolio Creator, explaining how the core objects of an issue hang together.

The complete stack of FutureFolio objects, from very largest to very smallest, is as follows:

  • One complete interactive edition of a magazine is called an issue, and, when it ships to readers, is provided as a single .zip file that contains all the text, graphics, movies and other media required to recreate the magazine on their devices.
  • Each issue is made up of one or more sections, such as News, Reviews and Tutorials. Sections are used to split up the table of contents and can also provide navigation around the issue.
  • Each section is made up of one or more pages, so the Tutorials section might have an iPhoto tutorial, a Photoshop tutorial, and a Google Chrome tutorial.
  • Each page is made up of one or more renditions, which define the way the page looks on a device. For example, you might create renditions for tablets in both portrait and landscape orientations.
  • Each rendition is made up of one or more layers, which are the actual items on the page – images, text, movies, and so on.

In practice, Future magazines use only one rendition: tablet portrait or tablet landscape.
