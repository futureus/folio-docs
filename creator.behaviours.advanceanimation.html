Advance Animation

If you have an image animation that is set not to advance automatically, this behaviour lets you advance it. This primarily used when the animation contains important information that should be shown until the user is ready to continue.

Animations that do not advance automatically can be advanced if the user taps the image, but if you want a dedicated picture on the page saying "Tap here to advance", use this behaviour.

Parameter: the name of the animation layer to advance.
