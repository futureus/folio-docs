Annotations

The annotation layer type consists of a background image with a set of information boxes pinned to different points of the image. When the user taps on an info box, the image is zoomed to a given point and zoom scale.

An annotation layer

The areas covered in this section are:

  1. Background Image and Scale - choosing and laying out the image to be annotated.
  2. Adding Pins - setting up your annotations.
  3. Zoom Options - controlling how your image zooms in response to user action.

To add an annotation layer, click on the Add annotation button.

Adding an annotation layer

Click on the layer settings button for your annotations layer to bring up the annotations dialog.

Annotation layer dialog
