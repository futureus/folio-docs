Anchor Points

By default, animations are anchored to their centre, so rotation (for example) will spin the object around its centre. This is fine for most purposes, but for advanced animations you might want to have a layer animate around a different point.

If you want to modify an animation's anchor point, please read the following:

  • The anchor point of a layer can only be changed inside an animation pre-run, and only once.
  • This means you should set it to the value you want using a rendition behaviour, then not touch it again.
  • Anchor points are specified as a value relative to the size of the layer: an X value of 0 means the left edge of the layer, an X value of 1 means the right edge of the layer, a Y value of 0 means the top of the layer, and a Y value of 1 means the bottom of the layer.
  • These relative points are used so that you don't have to worry about any scaling that has been applied to the layer: 0 and 1 refer to the edges of the layer regardless of whether it has been scaled.
  • You can use values beyond 0 and 1, and these allow you to rotate your layer from a more distant point. For example, rotating a layer that had an X anchor point of -10 would look like the layer was attached to a large wheel.

In the following example, the anchor point of the Picture1 layer is set to its top-left corner, then the layer is scaled up to 3x its normal size:

		
<animationBlock>
	<animation delay="0" duration="0">
		<layer name="Picture1" anchorX="0" anchorY="0" />	
	</animation>
	
	<animation delay="1" duration="3">
		<layer name="Picture1" scale="3" />
	</animation>	
</animationBlock>
