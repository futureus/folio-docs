Maps

Using Creator you can add fully interactive maps to your FutureFolio issues. You can display basic maps or overlay satellite imagery. You can let the user zoom or scroll the map, and you can add custom pins to specific points. Pins can have titles and interactivity behaviours added to them so that you for example can reveal a picture when the user taps a pin. Routes can be drawn on the map using the standard GPX format that most GPS devices can export.

Accessing the user's position

The user will be asked to consent to the App accessing the current geographical position. If the user does not consent, the map's functionality may be compromised.

Network connection

Map functionality requires the user's device to be connected to the internet in order to function fully. The map will not load if the user's device isn't online.
