Controlling the Page with Stories

You can show and hide layers as the user scrolls through a story layer by placing special markers in your text. These markers tell FutureFolio the name of the layer you want to show, and internally it uses a Show Single Layer effect (see "Show Single Layer") so that only one hidden layer is shown at a time.

You can place as many markers in your text as you want; FutureFolio will automatically find the one nearest the top of the page and show it.

Here's an example of marker layers in action:

<marker layer="layer1" />
Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nunc sed lorem magna.
Quisque id lectus et ligula commodo vestibulum. Nulla id eros sed urna ornare lobortis.
Vivamus tellus mauris, mollis in lobortis dignissim, vulputate quis leo. 
<marker layer="layer2" />
Duis pretium dapibus tristique. Etiam in mauris eu erat molestie convallis.
Duis convallis porta nibh, ut iaculis enim venenatis vitae.
Nullam ipsum leo, sodales et ornare id, dictum ac dui.
<marker layer="layer3" />
Vivamus sodales nisi quis odio rutrum id hendrerit metus dignissim.
Cras vitae leo a lacus vestibulum dapibus. Maecenas posuere, est sed molestie
ultricies, purus libero cursus neque, quis euismod orci magna vitae elit.

That example has three markers. As the reader scrolls down the page, first layer1 will be shown, then layer2, then layer3.

This technique is commonly used for pages that have long-form reading – you can show images that illustrate what the user is reading about right now rather than clutter the page up with many images.

It can also be used to detect when the user has started reading a story. You can show any layer you want using this system, so if, for example, your story was a review of a game in the App Store, you could make a button appear saying "Buy this on the App Store!" as soon as the user scrolls the story slightly, on the basis that scrolling the story means they are probably reading it.
