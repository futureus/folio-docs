Examples

A basic animation, to make a layer called Picture1 fade out over three seconds, looks like this:

<animationBlock>
	<animation delay="0" duration="3">
		<layer name="Picture1" alpha="0" />
	</animation>	
</animationBlock>

You could make it scale up as it fades out by adding a scale change, like this:

<animationBlock>
	<animation delay="0" duration="3">
		<layer name="Picture1" alpha="0" scale="3" />
	</animation>	
</animationBlock>

Movement can be specified either as an absolute value, like this:

<animationBlock>
	<animation delay="1" duration="1">
		<layer name="Picture1" x="250" />
	</animation>	
</animationBlock>

Or as a relative value, like this:

<animationBlock>
	<animation delay="1" duration="1">
		<layer name="Picture1" x="-75" />
	</animation>	
</animationBlock>

You can add multiple layer movements inside the same animation block, such as making one layer fade out and scale up while another fades in and scales down, like this:

<animationBlock>
	<animation delay="0" duration="3">
		<layer name="Picture1" alpha="0" scale="3" />
		<layer name="Picture2" alpha="1" scale="1" />
	</animation>	
</animationBlock>

Alternatively, if you want to affect multiple layers in the same way, just separate their names with commas, like this:

<animationBlock>
	<animation delay="0" duration="3">
		<layer name="Picture1, Picture2" alpha="0" scale="3" />
	</animation>	
</animationBlock>

If animations should be executed together, include them all in the same animation block. This animation will fade out and scale up Picture2 over three seconds, while also waiting two seconds and making Picture2 spin 180 degrees:

<animationBlock>
	<animation delay="0" duration="3">
		<layer name="Picture2" alpha="0" scale="3" />
	</animation>
	
	<animation delay="2" duration="3">
		<layer name="Picture2" angle="180" />
	</animation>	
</animationBlock>

It is important that you understand that these two animations will execute in parallel: the second one will begin while the first is still happening. The delay on the second animation is from the beginning of the animation script, rather than from the end of the first animation. If you want to have one animation wait for another to complete, see "Sequences".
