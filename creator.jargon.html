Glossary of Terms

  1. 360 - a set of photographs taken of an object from multiple angles that the user can spin around.
  2. Alpha - the level of opacity applied to an object or colour. A value of 1 means "fully visible" and a value of 0 means "fully invisible."
  3. Animation - a fixed image box that changes between various pictures over time.
  4. Chrome - the custom user interface that sits on top of your issue. This is designed to let the user navigate around the issue, and should not be
  5. CSS - the styling language used to format text.
  6. HTML - the formatting language used to mark up text.
  7. Hybrid - this describes an issue that is part digital replica and part interactive edition.
  8. Icon - the graphic that represents your title on users devices when it's shown alongside other apps. This is usually replaced by your latest cover.
  9. Layer - an object on a page, such as an image, text or a movie.
  10. Metadata - any text that describes your magazine and/or issue, such as its title, month, price, as well as a short description of what's in the issue.
  11. Newsstand - Apple's magazine delivery mechanism for iPhone and iPad.
  12. Particle system - a special effect that creates page effects such as smoke, fire and dust.
  13. Rendition - one layout of a page, targeting either tablet portrait, tablet landscape, phone portrait or phone landscape.
  14. Retina - Apple's name for high-resolution graphics. Retina screens (iPhone 4, iPhone 4s and iPad 3) are exactly 2x the size of non-retina screens.
  15. Scratch image - an image box that the user can touch to "scratch" away to reveal something below.
  16. Slideshow - a swipeable gallery of images.
  17. Splash screen - the full-screen logo that is shown when your app launches. This usually only appears for a second.
  18. Stack - a set of pages that are placed vertically above each other.
  19. Wipe image - an image box that the user can touch and swipe to reveal an image below.
  20. XML - the raw data format behind Creator issues. This XML is created for you as you work with the Creator tool.
