Edge Bounce

If you have enabled scrolling on your box layer, you also have the option of whether to enable edge bouncing, with the Enable edge bounce button.

When this is set to "ON", the user can drag the contents inwards from the edge. When they let go, it will bounce back to the edge.

When it is set to "OFF", the user can only drag the contents as far as the edge - they cannot over-scroll it.

Horizontal edge-bounce

Note that for standard pages, edge-bounce will only work vertically. When the user over-scrolls horizontally, this will be passed on to the page, eventually resulting in a page scroll. For custom-sized pages, horizontal edge-bounce is also possible.
