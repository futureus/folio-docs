Fit vs Stretch

Movie layer options

If the movie layer is of a different size than the movie that it will contain, Folio offers two different ways of fitting the movie content to the movie layer: Fit or Stretch. Fit will resize the movie as large as possible whilst still fitting inside the layer with maintained aspect ratio, whilst Stretch will as closely as possible fill the layer without respecting the aspect ratio. In general, if there is a big difference in dimensions between the layer and the movie itself, using Stretch will probably look odd. See "Movie layer options"
