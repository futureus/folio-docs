Show Single Layer

Although this is the most commonly used behaviour in FutureFolio, its use can be confusing so it has a dedicated discussion here.

Show Single Layer allows you to reveal exactly one hidden layer at a time. All layers that started the page hidden will be made hidden, except for the layers you specify as the parameter to this behaviour.

For example, if ImageA, ImageB and ImageC are hidden, using Show Single Layer and specifying "ImageA" will show only ImageA. If you have a second behaviour saying Show Single Layer with "ImageB" as its parameter, ImageA will be hidden and ImageB will be shown.

Using Show Single Layer ensures that no more than one hidden layer is revealed at a time. This is commonly used when you have multiple hidden layers on a page, and you don't want the user to be able to have them all visible simultaneously.

Parameter: the name of the layer to show. If you wish to show more than one simultaneously, separate their name with commas.

Autoshow Layers

Show Single Layers affect only layers that are marked as invisible when you are designing the page. So, if you want something to be affected by this behaviour and yet appear hidden when the user starts reading the page, type its name into the Autoshow Layers box under the Rendition Settings window.

Behind the scenes

Markers in Story layers use Show Single Layer to show and hide content.
