Adding a rendition

To add a rendition to a page, select that page in the issue editor window. Along the top above the page preview you will see four buttons, tap any grey button you want to add a rendition for. You will be asked if you want to add a blank page for this rendition or use a Template (Templates need to already have a page laid out in that rendition or it will just add a blank page).

Rendition buttons

Left - Active created rendition. Right - No rendition created, tapping this adds a rendition.
