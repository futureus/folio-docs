Page Scroll

By default, when a user scrolls some content inside a box layer, the scrolling is smooth and does not snap to position.

You may wish to use the Scroll in pages control to make the box contents scroll in a "paging" way. With this setting switched on, the box contents scroll in units of the box size - so if the user drags then lets go, the box contents will then dock to the nearest page (where the page size is the size of the box).

To see the paging scroll method in action, just scroll within the Photos app on iPhone / iPad.
