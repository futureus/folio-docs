Toggle Buttons

Toggle buttons allow you to create behaviours similar to those of radio buttons, they can also be used to control other layers on the page through a simple naming convention.

In its most basic form, you can create a toggle button by adding two pictures under the filenames field on the toggle settings panel. The first image specified will be used to represent the unchecked state of the toggle button and the second for the checked state. A third image is optional, if specified this image will be used to show a highlighted state of the toggle button.

Toggle buttons can be further customised in the following ways:

  1. Group name: by assigning a group name, e.g. "Group 1", toggle buttons within the same group will adopt the known behaviour of radio buttons. Only one toggle button within a group can be in the selected state at any one time, selecting one toggle button will deselect all the others in the same group.
  2. Sound effect: specify the filename of a sound sound file with the file extension. This sound file will then be played every time a toggle button is selected. If the toggle button is not assigned to a group then the sound will play when the toggle button is both selected and deselected.
  3. Initial state: this property dictates the state of the toggle button when it loads on the page. Setting this property to "ON" will show the image for the checked state of the toggle button. The default property is "OFF", if you leave all the toggle buttons within a group set to "OFF" Creator will automatically set the last toggle button in that group to "ON" when the page loads.

Using the naming convention "-on"

You can use toggle buttons to toggle the visibility of other layers on the page by naming layers accordingly. If we have a toggle button named "toggle-button" we can use the state of this toggle button to control the visibility of any other layer on the page named "toggle-button-on". Whenever "toggle-button" is selected "toggle-button-on" will be visible, deselecting "toggle-button" will hide "toggle-button-on". More complex situations can be created by combining this naming convention with the group name property, and nesting layers inside boxes etc.

When using the "-on" naming convention you must set the visibility of all the layers on the page named "layername-on" to hidden. The layer will then show according to the state of the toggle button.

Auto-show

If you use the auto-show layer feature to show a layer that is related to a toggle button you will over-ride the toggle feature for that layer.
