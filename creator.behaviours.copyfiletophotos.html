Copy a file to user photos

This behaviour allows you to copy one or more file from your issue into the user photos on the user's device. On iPad, iPhone and iPod Touch, this is the "Camera Roll" section of the Photos app, from where readers can set your picture as their device's wallpaper or take other actions with it.

Only files that you specify will be copied. If the file exists already, it will be copied again and will thus appear twice in the photos list.

Parameter: the filename in your issue that you want to copy. If you want to copy multiple files at the same time, you can specify them as a comma-separated list. It is recommended that you take this approach rather than use multiple behaviours that copy single files, because this latter option will show multiple messages to the user.
